# Figures

This folder contains paper-ready figures with shortened file names.

- `figure_1_main_snr_comparison.*`: main paper figure comparing SL, SSL, low-SNR exposure SSL, per-head SSL, and denoising SSL at label ratio `0.005`.
- `figure_2_sl_ssl_dense_snr.png`: baseline SL vs SSL dense SNR curve.
- `figure_3_per_class_recoverable_snr.png`: per-modulation accuracy in recoverable SNR.
- `figure_4_snr_regime_gain.png`: SSL gain summarized by SNR regime.

The main figure is provided as PNG, PDF, and SVG for paper/poster use.
